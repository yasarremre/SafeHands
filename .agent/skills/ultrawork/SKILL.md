## Ultrawork Principles

### 1. Aggressive Parallelization
- Spawn multiple agents for independent subtasks
- Never do sequentially what can be done in parallel
- Use the Task tool with multiple concurrent agents

### 2. Background Delegation
- Long-running tasks go to background agents
- Don't wait for slow tasks when other work can proceed
- Collect results asynchronously

### 3. Deep Exploration
- Thoroughly analyze before implementing
- Search docs, codebase, and patterns
- Use `librarian` (fast) and `explore` (thorough) agents

### 4. Relentless Execution
- Don't stop at "good enough"
- Complete the task fully
- Verify everything works
